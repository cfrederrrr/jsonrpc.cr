    Overview

    Client is, and can only be a wrapper for your favorite TCP/HTTP/S client It provides some convenience methods for handling method invocations, i.e. JSONRPC::Request and JSONRPC::Response but it is otherwise perfectly dumb

    This enables you to use whatever transport client you like, without having to deal much with the JSON RPC logic. However, it does mean that you will have to handle the transport logic yourself.
